# taro2.x RichText组件

## 组件特性

支持特性 | header 2
---|---
row 1 col 1 | row 1 col 2
row 2 col 1 | row 2 col 2

- h5端默认taro的基础组件rich-text，加了图片点击事件
- 小程序端自定义富文本渲染组件，支持音频、视频、图片、外部链接点击事件处理
